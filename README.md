## 📂 What's Inside

---

### 🦠 CovidData

A real-world dataset folder focused on the COVID-19 pandemic.

Contains vaccination rollout data, case statistics, and time-series records in **CSV and Excel formats**. These datasets are used as source data for projects across Excel, SQL, Power BI, and Tableau — making it a central reference point for the entire portfolio.

**Includes:**
- COVID-19 vaccination data by country and region
- Daily and cumulative case & death statistics
- Time-series pandemic trend records

---

### 🗄️ datasets

The fuel for every project in this portfolio.

A collection of datasets spanning multiple domains — healthcare, business, finance, and demographics. Each dataset is either used in an existing project or kept ready for future analysis work.

---

### 📊 Excel

The foundation of data analytics — and where this journey started.

This folder contains my Excel **learning notes and completed projects**, covering everything from basic formulas to full dashboard builds.

**Topics covered:**
- Data cleaning and transformation
- Pivot Tables and Pivot Charts
- VLOOKUP, INDEX-MATCH, and advanced formula logic
- Conditional formatting and data validation
- End-to-end dashboard projects with real datasets

---

### 📈 Power BI

Business intelligence brought to life through interactive dashboards.

Contains my **Power BI learning files and complete projects** built from raw data to polished reports.

**Topics covered:**
- Connecting and transforming data with Power Query
- Data modeling and table relationships
- DAX — measures, calculated columns, KPIs
- Interactive dashboards with slicers, filters, and drill-throughs
- End-to-end report design and layout

---

### 🛢️ SQL

Where data gets questioned — and answers come from logic.

SQL projects developed across two platforms:

- **MySQL Workbench**
- **SQL Server Management Studio (SSMS)**

**Topics covered:**
- Core queries — SELECT, WHERE, GROUP BY, HAVING, ORDER BY
- All JOIN types — INNER, LEFT, RIGHT, FULL OUTER
- Subqueries, CTEs, and nested logic
- Window functions — ROW_NUMBER, RANK, DENSE_RANK, PARTITION BY
- Data cleaning and standardization using SQL
- Real-world case-study analysis projects

---

### 🎨 Tableau

Data storytelling — turning numbers into narratives.

My **Tableau learning files and published dashboards**, built to communicate insights clearly and visually.

**Topics covered:**
- Connecting Tableau to CSV, Excel, and database sources
- Building interactive, filterable dashboards
- Calculated fields, parameters, and sets
- Visual best practices for data storytelling
- Real-world dashboards using public datasets

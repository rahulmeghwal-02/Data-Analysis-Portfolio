---

## 📁 Projects

### 1. 🏠 AirBnB Project
**Folder:** `AirBnB Project - 1/`

An end-to-end data analysis of the **AirBnB Seattle 2016** dataset, visualized through Tableau dashboards.

**Key Insights Explored:**
- Average price per bedroom
- Revenue trends across the year
- Price distribution by ZIP code
- Distinct bedroom listing counts

**Tools Used:** Tableau Desktop  
**Dataset:** AirBnB Seattle Open Data (2016)

---

### 2. 🎬 Netflix Project
**Folder:** `Netflix ProjectA - 2/`

A comprehensive visual analysis of the **Netflix content library**, exploring trends in content type, genre, ratings, and global distribution.

**Key Insights Explored:**
- Movies vs. TV Shows distribution
- Content added over the years
- Top genres and categories
- Country-wise content breakdown
- Ratings distribution

**Tools Used:** Tableau Desktop  
**Dataset:** Netflix Titles Dataset (Kaggle)

---

### 3. 🧪 Practice Workbook
**File:** `Practice_Tableau.twbx`

A Tableau Packaged Workbook used for hands-on practice and skill building.  
Contains experimental visualizations, chart types, and calculated fields.
